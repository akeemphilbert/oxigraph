# Oxigraph documentation

This is the home for Oxigraph's in-repo documentation. Pick the area that matches
what you're here to do:

- **[Users](users/README.md)** — you want to *use* Oxigraph: install it, run the
  server, and query your data from the command line or from Rust, Python, or
  JavaScript.
- **[Contributors](contributors/README.md)** — you want to *change* Oxigraph: set up
  a development environment, understand how the codebase fits together, and ship
  your first change.

Each area is organized the same way:

| Section | Use it when… |
| ------- | ------------ |
| Tutorials | you're new and want to learn by doing — follow the steps end to end |
| How-to guides | you have a specific task and want a recipe for it |
| Reference | you need to look up a fact — commands, options, structure |
| Explanation | you want to understand how or why something works the way it does |
