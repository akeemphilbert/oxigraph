# Reference

The authoritative API references live with each released package — they are
generated from the source and always match the release you installed:

| Package | Reference |
| ------- | --------- |
| `oxigraph` (Rust crate) | [docs.rs/oxigraph](https://docs.rs/oxigraph) |
| `pyoxigraph` (Python) | [pyoxigraph.readthedocs.io](https://pyoxigraph.readthedocs.io/) |
| `oxigraph` (npm) | [package README on npm](https://www.npmjs.com/package/oxigraph) |
| `oxigraph-cli` (server & CLI) | [CLI README](../../../cli/README.md) and `oxigraph --help` |
| Docker image | [ghcr.io/oxigraph/oxigraph](https://github.com/oxigraph/oxigraph/pkgs/container/oxigraph) |

The server's HTTP interface follows the W3C
[SPARQL Protocol](https://www.w3.org/TR/sparql11-protocol/) and
[Graph Store Protocol](https://www.w3.org/TR/sparql11-http-rdf-update/)
specifications.
